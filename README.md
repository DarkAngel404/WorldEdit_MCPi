# WorldEdit_MCPi

Normally to re-create WorldEdit mod in MCPi (Reborn or ++)

# Installation

In a terminal :
`git clone https://github.com/DarkAngel404/WorldEdit_MCPi.git`

### Don't forget to modify *TheRunaway* by your username (from line 94 to line 99 of `main.py`)

# Launch

Just launch `main.py`
